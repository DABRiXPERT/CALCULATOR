# 使用說明
其實就照一般計算機使用的方式而已。  
按鍵可以連按，按下等於鍵後的所有操作必須清空(C/AC)才可以進行新的運算，否則會繼續上一次的結果。  
輸入限制最大為 <b>15</b> 位數，超過會跳出警告視窗。
